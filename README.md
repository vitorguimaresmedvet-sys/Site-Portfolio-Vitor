## 🇧🇷 Português

### 📝 Sobre o Projeto
Este é o meu portfólio pessoal, desenvolvido durante as aulas do **SCTec (SENAI)**. O projeto foi construído para centralizar minha trajetória profissional, meus projetos de desenvolvimento web e minhas criações em modelagem 3D.

O site é totalmente responsivo e serve como uma vitrine para a minha transição de carreira: de **Médico Veterinário** para **Desenvolvedor e Artista 3D**.

### 🚀 Funcionalidades
- **Navegação Suave:** Âncoras que facilitam a navegação entre as seções.
- **Menu Responsivo:** Menu hambúrguer funcional para dispositivos móveis via JavaScript.
- **Cards de Projetos:** Exibição clara de competências em Web Dev e Blender.
- **Formulário de Contato:** Interface pronta para captação de mensagens.
- **Links Externos:** Integração com LinkedIn, GitHub, ArtStation e MakerWorld.

### 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização moderna com Flexbox e Media Queries.
- **JavaScript (Vanilla):** Lógica para interatividade do menu.
- **Blender:** (Assets mencionados nos projetos).
